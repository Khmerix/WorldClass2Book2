=====================================================
⚠️ IMPORTANT NOTES FOR SHARING
=====================================================

1. KEEP ALL FILES TOGETHER: The system requires all files
   to be in their current folder structure to work properly.

2. VIDEOS ARE INCLUDED: All 6 unit videos (MP4) are included
   in the package for offline viewing.

3. AUTHENTICATION: All content pages check for login status.
   If someone tries to access files directly without logging
   in first, they will be redirected to the login page.

4. SESSION SECURITY: Login status is stored in browser
   session storage and will be cleared when the browser
   is closed.

5. CHANGING PASSWORDS: To change passwords, edit login.html
   and look for the validUsers object in the JavaScript.

=====================================================
🎥 VIDEO TROUBLESHOOTING
=====================================================

If videos don't play:
1. Make sure you're using book2b_complete.html after login
2. Check that MP4 files are in BOTH locations:
   - Book2Complete/ folder
   - Book2Complete/files seperate/ folder
3. Try opening test_video.html to verify video files
4. Use a modern browser (Chrome, Firefox, Edge, Safari)

=====================================================
🔒 SECURITY RECOMMENDATIONS
=====================================================

1. Change default passwords before sharing with other teachers
2. Share the login credentials separately from the files
3. Remind teachers to logout when finished
4. For maximum security, host on a password-protected server
   instead of sharing files directly
